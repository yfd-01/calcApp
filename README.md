# calcApp
A calculator running on the Windows platform using PyQt5 to implement GUI

## 使用
双击calc.exe应用程序即可运行

## 特别处理和检错功能
* 支持负数运算，能对负号和减号进行判断 如：`2x-3`
* 能对冗余括号和符号进行判断，得出可计算式子 如：<br>`(((8+6)x(-2` => `(((8+6)x(-2)))` 、 `(88)+((((2x3/-2)+-` => `(88)+((((2x3/-2))))`
* 若为无效表达式，则捕捉进行错误提示 如：`((-`
